Help & Documentation

Adding a New Customer:
  1. Navigate to Customer Management section
  2. Click "Add New Customer" button
  3. Fill in all required fields (Account Number, Name, Address, Phone)
  4. Click "Save Customer" to add to the system
Editing Customer Information:
  1. Find the customer in the customer list
  2. Click the "Edit" button in the Actions column
  3. Update the necessary information
  4. Click "Update Customer" to save changes

Adding New Items:
  1. Go to Item Management section
  2. Click "Add New Item" button
  3. Enter item details (Code, Name, Category, Price, Stock)
  4. Click "Save Item" to add to inventory
Managing Stock:
  1. Use the search function to find specific items
  2. Edit items to update stock quantities
  3. Monitor stock levels in the inventory table

Creating a Bill:
  1. Navigate to Billing System
  2. Select a customer from the dropdown
  3. Add items by selecting them and specifying quantities
  4. Review the bill summary
  5. Click "Generate Bill" to create the final bill
Bill Calculations:

The system automatically calculates subtotal, tax (10%), and total amount. All calculations are updated in real-time as you add or remove items.
